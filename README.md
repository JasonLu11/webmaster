# XRK网站

一个简单的单页面网站，以XRK.jpg为背景图片，点击时自动跳转到指定下载链接。

## 功能特点

- 全屏背景图片显示
- 点击页面任意位置跳转到下载链接
- 响应式设计，适配不同屏幕尺寸
- 悬停效果和交互提示

## 文件结构

```
├── index.html      # 主页面文件
├── XRK.jpg         # 背景图片
└── README.md       # 项目说明
```

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击页面任意位置即可跳转到下载链接

## 技术栈

- HTML5
- CSS3
- JavaScript

## 部署

可以直接部署到GitHub Pages或其他静态网站托管服务。
